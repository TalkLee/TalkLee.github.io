Hexo 常用命令

hexo init [folder] //生成hexo博客

hexo new [layout] //生成新的静态页面,layout默认使用 _config.yml.

hexo generate //生成静态文件

hexo server //运行服务器，默认4000为端口号

hexo deploy //部署到远程站点
