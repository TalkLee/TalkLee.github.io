具名函数的多种调用方式

  • 每日一博

背景代码:

function foo(){
    alert('a');
}

方法1:

括号()运算符调用

foo();

方法2&3:

call和apply方法

foo.call();  //call和apply中传入的第一个参数为函数调用中作用域(this对象),
单纯为调用函数的话，传入变量常量任何均可
foo.apply();

方法4：

new构造函数调用方法

new foo()

注意以下情况:

    function foo(param){
    alert(param);
    return param;   //有返回值
}

var a = new foo({ b : '1'});  //返回传入对象
var a = new foo('OK');    //返回另一个对象

若函数声明中返回对象，则构造函数调用时返回指定的对象
若函数声明中返回基本类型(包括null和undefined),构造函数调用时实际返回函数语句中生成的对象(通过this给生成的对象添加属性)

function foo(param){
    alert(param);
    this.b = '1';
    return null;
}

上例中return语句无效,返回对象，包含属性b.

若函数声明中声明新对象，并且通过this给构造函数生成对象添加属性，又会出现什么情况呢?

若无返回值或者返回基本类型或者返回this，则返回this产生的对象，
若返回构造函数中声明的对象，则返回指定的对象(工厂模式的写法)

function foo(param){
    this.pro = 'this';
    var obj = {};
    obj.name = 'foo';
    obj.msg = param;
    //return this;  返回对象中包括pro属性
    //return obj;  返回对象为obj
}

工厂模式:

function foo(param){
    var obj = {};
    obj.msg = param;
    return obj;
}
var a = new foo('2');  //构造函数方式调用
var b = foo('3');  //函数调用,两种方法返回结果相同，原因上面已经讨论过了
